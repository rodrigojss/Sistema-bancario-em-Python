# Desafios - Projeto em Python DIO

Um desafio de nével básico proposto pela plataforma [Dio.](https://www.dio.me/) 

> ## Um projeto de um Sistema Bancário com Python



Neste projeto, haveria a necessidade de criar as funções de "Depósito", "Saque", "Extrato" e a opção de cadastrar contas e cadastrar usuários, além de permitir a listagem das contas existentes no sistema.
